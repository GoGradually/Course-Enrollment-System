또한 IMPLEMENTATION_CHECKLIST.md 파일 내에 이 프로젝트를 구현하기 위한 상세 체크리스트가 작성되어 있다.

이를 바탕으로, "E. 초기 데이터 생성"을 완성하라.

---

또한, 완성된 요구사항은 IMPLEMENTATION_CHECKLIST.md 파일 내 체크박스에 반영되어야 한다.

---

• 초기 데이터에서 수강신청(Enrollment)까지 함께 생성할까요?
answer: 생성하지 않음 (Recommended)
• Agent A와 동시 작업 중인데 체크리스트 반영은 어떻게 진행할까요?
answer: 체크리스트 전체 동기화

1. 단일 학기 가정 유지.
2. 초기 Enrollment는 생성하지 않는다.
3. 시드는 결정론적(randomSeed 고정)으로 재현 가능하게 한다.
4. 테스트 기본 환경에서는 대규모 시드를 비활성화한다.
5. 동시성 제어 로직/엔드포인트는 Agent A 영역으로 간주하고 본 작업에서 변경하지 않는다.
